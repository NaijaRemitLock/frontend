# Security Policy

## Reporting a Security Vulnerability

If you discover a security vulnerability in NaijaRemitLock, please **DO NOT** create a public issue.

Instead, please email us at: **calebbola45@gmail.com**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Your name and contact info (optional)

We will respond within 24 hours and work with you to fix the issue before public disclosure.

## Security Updates

We monitor and update dependencies regularly using Dependabot.

### Supported Versions

| Version | Status | Support Until |
|---------|--------|----------------|
| 1.0.x   | ✅ Supported | Current |
| < 1.0   | ❌ Unsupported | Not supported |

## Security Best Practices

As a user of NaijaRemitLock:
- ✅ Keep your Freighter wallet secure
- ✅ Never share your private keys
- ✅ Always verify the website URL
- ✅ Use HTTPS connections only
- ✅ Report suspicious activity immediately

## Our Security Measures

- Dependabot enabled for vulnerability alerts
- Secret scanning enabled
- Code security analysis enabled
- Pull request reviews required
- Branch protection rules enforced
