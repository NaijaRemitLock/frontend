# NaijaRemitLock - Frontend UI

Web interface for purpose-locked remittances on Stellar.

## What it does

Users connect their Freighter wallet, create lockboxes for specific purposes (school fees, rent, business, hospital), and view their lockboxes.

## Features

- Connect Freighter wallet
- Create purpose-locked lockboxes
- View your lockboxes
- Demo mode for testing

## How to use

1. Install Freighter wallet extension from stellar.org/freighter
2. Open index.html in your browser
3. Click "Connect Freighter Wallet"
4. Create a lockbox with purpose, amount, and release date
5. View your lockboxes below

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Freighter API
